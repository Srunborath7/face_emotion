About Emotion Detector

The Emotion Detector is a real-time facial expression recognition system powered by computer vision and deep learning. It captures video from a live webcam feed, processes facial features, and predicts the user's emotion using a trained model.

Built using Python, Flask, OpenCV, and Bootstrap 5, this application aims to demonstrate the use of artificial intelligence in enhancing human-computer interaction.
